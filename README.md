# FindIT

## Description

FindIT is a web application that allows you to search for a startups. The application is based on the NestJS and Next.js frameworks.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development (watch mode)
$ make start-dev

# production mode
$ make start
```

## Test

```bash
# unit tests
$ make tests

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
